What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a structure that represents a web page as a tree of objects. Each part of the HTML (like elements, text, and attributes) becomes an object. JavaScript can then use the DOM to interact with and change the content or style of the webpage.

How do you select on element from the DOM?

To select elements from the DOM: 1/ By ID: document.getElementById('id') 2/ By class: document.querySelector('.class') (selects the first match) 3/ By tag: document.querySelector('tag') 4/ By multiple selectors: document.querySelectorAll('selector') (for all matching elements)

What is event delegation in the context of teh DOM, and why is it useful?

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, use element.setAttribute() to change attributes like src or href. For styles, use element.style.property (e.g., element.style.color = 'blue';) to change the CSS properties directly. Both methods update the element dynamically.
